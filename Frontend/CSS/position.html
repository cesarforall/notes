Position

  • relative. Su posición se determina según su orden en el elemento contenedor
  • absolute. Su posición se determinará a partir del elemento contenedor relativo más cercano
  • fixed. El elemento se queda fijo en la página
  • sticky. Se queda en la parte superior de la pantalla mientras se hace scroll hasta que llega un próximo elemento del mismo tipo y con las mismas reglas css (hermano). Es necesario definir top
  • static. Es la posición por defecto
  • initial. Es la posición inicial de un elemento
  • inherit. Hereda la posición del elemento padre
